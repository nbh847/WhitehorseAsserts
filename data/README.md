# 股票数据本地存储说明

## 功能介绍

这个文件夹用于存储您的股票数据。由于浏览器的安全限制，应用程序无法直接将数据保存到这个文件夹，但您可以通过以下步骤手动将数据保存到这里：

## 如何备份数据到本地文件夹

1. 在应用程序的"白马股"标签页中，点击**导出数据**按钮
2. 在浏览器的下载对话框中，选择保存位置为当前的 `data` 文件夹
3. 点击保存，数据将以 JSON 格式保存到本地

## 如何从本地文件夹恢复数据

1. 在应用程序的"白马股"标签页中，点击**导入数据**按钮
2. 在文件选择对话框中，导航到当前的 `data` 文件夹
3. 选择您之前备份的 JSON 文件
4. 点击打开，数据将被导入到应用程序中

## 数据文件格式

数据以 JSON 格式保存，文件名格式为：`blueChipStocks_YYYY-MM-DD.json`

## 最佳实践

1. 定期导出数据到这个文件夹，建议每周至少一次
2. 可以创建多个备份文件，使用不同的日期或版本号
3. 如果更换浏览器或端口，可以通过导入功能快速恢复数据

## 示例文件

首次使用时，这个文件夹是空的。当您导出数据后，将看到类似以下的文件：
- `blueChipStocks_2025-12-20.json`
- `blueChipStocks_2025-12-27.json`

## 注意事项

- 不要修改 JSON 文件的格式，否则可能导致导入失败
- 确保备份文件包含所有必要的字段：股票代码、名称、买点、卖点、买入股数、当前股价等
- 如果您需要在不同设备间同步数据，可以将这个文件夹中的备份文件复制到其他设备
